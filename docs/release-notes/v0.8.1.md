# v0.8.1 — Round-2 code-review follow-ups

> 2026-05-29 · 1 P1 design + 6 P2 mechanical fixes · 437 / 437 tests passing

Round-2 follow-ups to v0.8.0's retroactive code review pass. Closes the last
P1 (prose-marker injection design) and six P2 mechanical fixes that landed
together rather than being deferred to a future session.

## 中文

### Highlights

| 维度 | 改动 | 为什么重要 |
|---|---|---|
| **adv-001 防御落地 (P1)** | `core/ambient-fence.mjs` 加入 managed-header sentinel：`<!-- nexel-managed-fence v1; do not edit between markers -->` 作 fence body 首行；replace 前校验 sentinel 在场 | v0.8.0 retroactive review 唯一未闭合 P1。用户散文若含字面 marker，refuse 覆盖 (live-verified collision class 关闭) |

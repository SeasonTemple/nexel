# v0.8.4 — Round-5 backlog drain + ADR-0014 envelope unification

> 2026-05-29 · 4 deferred backlog items closed + 2 P1s caught in pre-commit review · 455 / 455 tests passing (+11 hook integration tests)

Round-5 closes the 4 honest-doable items from v0.8.3's deferred backlog
(#1 lockfile race, #3 envelope unification, #4 --target naming, #7
pre-push hook integration test). The pre-commit review (mandatory per
ADR-0013) caught 2 P1 issues introduced by the round-5 fixes themselves
and they were fixed before commit.
